# Proyecto de Prácticas: Curso de Programación con Python

Este repositorio contiene una serie de funciones desarrolladas como parte del curso de Programación con Python ofrecido por la Universidad de los Andes a través de Coursera. Estas funciones han sido diseñadas para practicar conceptos clave enseñados en el curso y pueden ser utilizadas para comprender y aplicar habilidades básicas de programación en Python.

## Descripción

El propósito de este repositorio es proporcionar a los estudiantes un conjunto de funciones básicas que les permitan practicar y aplicar los conceptos aprendidos durante el curso. Estas funciones han sido desarrolladas siguiendo las instrucciones y los problemas planteados a lo largo de las lecciones.
